# FastNotes -Flutter

Dette prosjektet er en enkel notatapplikasjon utviklet med Flutter. 

## Funksjonalitet:
- Vise liste over notater
- Opprette nytt notat
- Vise detaljer for notat 
- Navigasjon mellom skjermer

## Krav: 
- Flutter installert

## Hvordan kjøre prosjektet
1. Gå til prosjektmappe 
   cd fastnotes_flutter
2. Installer dependencies: flutter pub get
3. Start appen: flutter run

Appen lagrer ikke i minnet, ingen permanent lagring
